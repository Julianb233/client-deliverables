# Client Deliverables Portal

Auto-generated portal for client deliverables powered by AI Acrobatics Orchestra.

## Structure

```
clients/
├── {client-slug}/
│   ├── index.html          # Client portal page
│   ├── README.md           # Client overview
│   └── deliverables/
│       └── {date-title}/
│           ├── README.md   # Deliverable details
│           └── files...
```

## How It Works

1. **Agents create content** (scripts, guides, assets)
2. **Derek-Deliverables publishes** to this repo
3. **GitHub Pages auto-deploys** the portal
4. **Clients access** their branded portal
